Instrucciones para cargar el archivo CSV

Para que la aplicación funcione correctamente, necesitas cargar el archivo evangelio.csv. Puedes hacerlo de dos maneras:

  1. Usar un servidor web local (recomendado):
    • Si usas VS Code, instala la extensión "Live Server" y haz clic en "Go Live"
    • Con Python: abre una terminal en la carpeta y ejecuta python -m http.server
    • Con Node.js: instala http-server con npm install -g http-server y ejecuta http-server
  2. Cargar manualmente el archivo:
    • Haz clic en el botón "Cargar archivo evangelio.csv" y selecciona el archivo desde tu computadora

El archivo CSV debe tener el siguiente formato:

fecha,cita,resumen
2025-10-01,"Lc 9,57-62","Resumen del evangelio..."
2025-10-02,"Lc 10,1-12","Resumen del evangelio..."
